# GITS Pages Component

> Reusable Pages Component

## Installation

npm

```
npm i @gits-id/pages
```

yarn

```
yarn add @gits-id/pages
```

pnpm

```
pnpm i @gits-id/pages
```

## Usage

```vue
<script setup lang="ts">
// import styles
import '@gits-id/pages/dist/style.css';
// import component
import Pages from '@gits-id/pages';
</script>

<template>
  <Pages />
</template>
```

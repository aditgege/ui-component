# GITS File Input Component

File Input Component.

## Installation

npm

```
npm i @gits-id/file-input
```

yarn

```
yarn add @gits-id/file-input
```

pnpm

```
pnpm i @gits-id/file-input
```

## Usage

```vue
<script setup lang="ts">
import FileInput from '@gits-id/file-input';
</script>

<template>
  <FileInput />
</template>
```

# GITS AppBar Component

> Application Bar Component

## Installation

npm

```
npm i @gits-id/app-bar
```

yarn

```
yarn add @gits-id/app-bar
```

pnpm

```
pnpm i @gits-id/app-bar
```

## Usage

```vue
<script setup lang="ts">
import AppBar from '@gits-id/app-bar';
</script>

<template>
  <AppBar />
</template>
```

# Shimmer

Shimmer component AKA loading animation / content placeholder.

## Installation

npm

```
npm i @gits-id/shimmer
```

yarn

```
yarn add @gits-id/shimmer
```

pnpm

```
pnpm i @gits-id/shimmer
```

## Usage

```vue
<script setup lang="ts">
import Shimmer from '@gits-id/shimmer';
</script>

<template>
  <Shimmer />
</template>
```

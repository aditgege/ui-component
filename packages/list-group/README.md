# GITS Alert Component

> Alert Component

## Installation

npm

```
npm i @gits-id/alert
```

yarn

```
yarn add @gits-id/alert
```

pnpm

```
pnpm i @gits-id/alert
```

## Usage

Import `Alert` to your component:

```vue
<script setup lang="ts">
import Alert from '@gits-id/alert';
</script>

<template>
  <Alert color="primary">Lorem ipsum</Alert>
</template>
```

# GITS Autocomplete Component

Autocomplete Component.

## Installation

npm

```
npm i @gits-id/autocomplete
```

yarn

```
yarn add @gits-id/autocomplete
```

pnpm

```
pnpm i @gits-id/autocomplete
```

## Usage

```vue
<script setup lang="ts">
import Autocomplete from '@gits-id/autocomplete';
</script>

<template>
  <Autocomplete />
</template>
```

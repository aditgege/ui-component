# GITS Form Select Component

Form Select Component.

## Installation

npm

```
npm i @gits-id/form-select
```

yarn

```
yarn add @gits-id/form-select
```

pnpm

```
pnpm i @gits-id/form-select
```

## Usage

```vue
<script setup lang="ts">
import FormSelect from '@gits-id/form-select';
</script>

<template>
  <FormSelect />
</template>
```

# GITS Logo Component

> Reusable Logo Component

## Installation

npm

```
npm i @gits-id/logo
```

yarn

```
yarn add @gits-id/logo
```

pnpm

```
pnpm i @gits-id/logo
```

## Usage

```vue
<script setup lang="ts">
// import styles
import '@gits-id/logo/dist/style.css';
// import component
import VLogo from '@gits-id/logo';
</script>

<template>
  <VLogo />
</template>
```

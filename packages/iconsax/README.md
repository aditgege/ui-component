# Vue Iconsax

## Installation

npm

```
npm i @gits-id/iconsax
```

yarn

```
yarn add @gits-id/iconsax
```

pnpm

```
pnpm i @gits-id/iconsax
```

## Usage

```vue
<script setup lang="ts">
import Iconsax from '@gits-id/iconsax';
</script>

<template>
  <Iconsax name="airplane" />
</template>
```

# GITS UI

Free and open-source UI Component for Vue 3.

## Installation

npm

```
npm i @gits-id/ui
```

yarn

```
yarn add @gits-id/ui
```

pnpm

```
pnpm i @gits-id/ui
```

## Usage

```ts
// file: main.ts

import GitsUI from '@gits-id/ui';
import '@gits-id/ui/styles';
// or load SCSS styles
// import '@gits-id/ui/styles.scss';

const app = createApp(App);

app.use(GitsUI);
```

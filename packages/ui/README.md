# GITS UI Component

## Installation

npm

```
npm i @gits-id/ui
```

yarn

```
yarn add @gits-id/ui
```

pnpm

```
pnpm i @gits-id/ui
```

## Usage

```ts
// main.ts

import GitsUi from '@gits-id/ui';
// gits ui styles
import '@gits-id/ui/style.css';

const app = createApp(App);

app.use(GitsUi); // use plugin
```

# GITS Navbar Component

> Reusable Navbar Component

## Installation

npm

```
npm i @gits-id/navbar
```

yarn

```
yarn add @gits-id/navbar
```

pnpm

```
pnpm i @gits-id/navbar
```

## Usage

```vue
<script setup lang="ts">
// import styles
import '@gits-id/navbar/dist/style.css';
// import component
import Navbar from '@gits-id/navbar';
</script>

<template>
  <Navbar />
</template>
```

# GITS VModal Component

> Reusable VModal Component

## Installation

npm

```
npm i @gits-id/modal
```

yarn

```
yarn add @gits-id/modal
```

pnpm

```
pnpm i @gits-id/modal
```

## Usage

```vue
<script setup lang="ts">
// import styles
import '@gits-id/modal/dist/style.css';
// import component
import VModal from '@gits-id/modal';
</script>

<template>
  <VModal />
</template>
```

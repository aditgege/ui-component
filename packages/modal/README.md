# GITS Modal Component

> Reusable Modal Component

## Installation

npm

```
npm i @gits-id/modal
```

yarn

```
yarn add @gits-id/modal
```

pnpm

```
pnpm i @gits-id/modal
```

## Usage

```vue
<script setup lang="ts">
import Modal from '@gits-id/modal';
</script>

<template>
  <Modal />
</template>
```

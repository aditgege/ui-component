# GITS Form Group Component

Form Group Component.

## Installation

npm

```
npm i @gits-id/form-group
```

yarn

```
yarn add @gits-id/form-group
```

pnpm

```
pnpm i @gits-id/form-group
```

## Usage

```vue
<script setup lang="ts">
import FormGroup from '@gits-id/form-group';
</script>

<template>
  <FormGroup />
</template>
```
